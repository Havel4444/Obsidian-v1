### ANNOTATIONS
##### COMANDOS:

Muestra la ruta ubicada.
```
pwd
```

**Muestra la ubicacion de un archivo ejecutable**.
```
which ls
```
**Muestra la ubicacion del valor actual**
Que es la ruta completa del shell en uso.
```
echo $SHELL
```

**==SHELLS EXISTENTES==**
```
cat /etc/shells
```
**==COMANDO PARA CAMBIAR DE SHELL==**
bash, zsh, pwsh y mas.
```
chsh -s /bin/bash
chsh -s /bin/zsh
chsh -s /bin/pwsh
```
